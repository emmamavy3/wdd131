Overview

Purpose

I want to make a site that will allow me to view all my favorite movies and books. It should also help me receive recommendations from others. This website will also eventually help others keep track of their favorite books and movies.

Audience

I am the main audience especially at the start but I would like to make it a site where people can share recommendations. That would make it so movie and book lovers can also have a place to record their favorite books and movies.

Dynamic elements

I will implement a dark mode toggle using JavaScript and dynamically display books and movies by fetching data from a database or user submissions.
